# Chat online

### Como rodar o projeto

- Instale o projeto backend e siga as instruções que estão no readme do backend
- Instale esse projeto em sua máquina
- Instale as dependencias com o comando:
```shell
yarn
```
- Rode o comando para iniciar o projeto localmente:
```shell
yarn start
```
- Importante:
  - O projeto permite apenas 1 usuário logado por navegador.
